## MarkdownOffice 👋

Official Website: [markdownoffice.com](https://markdownoffice.com)

MarkdownOffice is an LLM-first, Markdown-native office suite where documents, spreadsheets, and presentations are created as structured text, not binary files.
Built for AI generation, collaboration, and version control, it treats Markdown as the single source of truth and renders it into enterprise-grade documents - fully self-hosted, auditable, and open by design
